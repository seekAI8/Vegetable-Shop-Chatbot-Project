# Muhammad Akram Vegetable Shop Chatbot

## Project Description

This is a simple console-based Vegetable Shop Chatbot written in Python.
It allows users to: - View the vegetable price list - Check the price of
a vegetable - Buy vegetables - Calculate the total bill - Exit the
program

## Requirements

-   Python 3.x

## How to Run

1.  Download `Muhammad_Akram_Vegetable_Shop_Console.py`
2.  Open a terminal or Command Prompt.
3.  Run:

``` bash
python Muhammad_Akram_Vegetable_Shop_Console.py
```

## Sample Vegetables

-   Potato
-   Tomato
-   Onion
-   Carrot
-   Cabbage
